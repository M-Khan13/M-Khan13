### Things I've built

**[Café Ops](https://github.com/M-Khan13)** &nbsp;—&nbsp; `React` `Node` `Express` `MongoDB` `Gemini`

A two-sided operations app for cafés. JWT auth with role-based access control, orders modeled as a state machine rather than a status string, and AI-generated staff tasks that pass through a five-check validation layer before they're allowed near the database. Solo build, backend to deploy.

**[RAG Document Q&A](https://github.com/M-Khan13)** &nbsp;—&nbsp; `Node` `Express` `MongoDB` `Ollama` &nbsp;·&nbsp; *in progress*

Document question-answering built from scratch — no LangChain, no LlamaIndex, because understanding each piece was the point. PDF chunking with page metadata, embeddings via `nomic-embed-text`, hand-rolled cosine-similarity retrieval, and answers that cite their source or admit they don't know.
